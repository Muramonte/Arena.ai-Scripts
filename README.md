chat export old отличие от новой:
- кнопка внизу по центру "Export"
- поддержка заголовков `h1`–`h6` (включая короткие, вроде `#1.1`)
- заголовки не режутся фильтром минимальной длины
- заголовки сериализуются в markdown (`#`, `##`, ...)
- исправлено дублирование текста (контейнер + вложенные блоки) — теперь после сбора отбрасываются элементы-контейнеры, если внутри них уже есть отдельно собранные блоки
- нумерованные списки (`<ol>`) сохраняют номера, а не превращаются в `-`
- пустые ссылки (например, иконка без текста) больше не теряют `href`
- убран мёртвый код (`sleep()`, неиспользуемая константа `HOTKEY`)
- добавлен стабильный порядок сортировки (индекс при сборе) как дополнительный тай-брейкер, чтобы при почти одинаковых координатах порядок не был случайным
  
- стоит `@match` только для arena.ai. Можно поставить `// @match        *://*/*` чтобы скрипт работал на любом сайте.
